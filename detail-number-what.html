NumberWhat

Discalculia is a difficulty in learning or comprehending arithmetic, such as difficulty in understanding numbers, learning how to manipulate numbers, and learning facts in mathematics. It can also refer to a difficulty in reading a standard clock based on numbers.NumberWhat is a solution designed specificaly for people with discalcular disfunction.

Github repository

Inspiration

Katrine, the designer of our team, is discalcular and has difficulties telling the time on a regular watch. She had an idea that would make it easier for her and others with discalculia to tell the time.

What it does?

The clock is shaped like a pie chart with two colors, a background color and front color. Instead of minute numbers, the front color indicates the time that has passed and the background color indicates the time that is coming. When the minutes of the hour is from 0-29 minutes, the clock shows the hour number of the time passed. But when the minutes of the hour is 30-59 minutes the clock shows the hour number of the time that is coming.

How have we built it?

The projects has been realised for three different platforms:

  1. Pebble Watch - with use of pebble-cloud sdk - Javascript - Rocky.js library - Repository
  2. Android Widget - native java application with use of Android Studio
  3. Android wear watchface - native java application with use of Android Studio
