P2P Social Network

For those who have seen the Sillicon Valley tv series a notion of distributed internet shall be familiar. The idea for a distributed p2p social network project was born out of this notion.

Used technologies

  1. Javascript
  2. Typescript
  3. Node.JS
  4. AngularJS

Github repository

The application development is currently in progress.

The p2p social network is going to be built on top of Kademlia p2p routing table. The primary focus of the project are the security, privacy, and performance of distributed systems. We will try to answer the question if it would be possible to substitute a centralized system like Facebook with fully distributed one?
